## tabulate（赛事排表）

- 本插件基于 Yunzai-Bot-V3 运行
- 联系作者：RetuEase（QQ：3573414538）

#### 安装

> 在云崽根目录下

```shell
> git clone https://github.com/RetuEase/tabulate.git ./plugins/tabulate/
```

```shell
> git clone https://gitee.com/smallcui/tabulate.git ./plugins/tabulate/
```

#### 使用

##### 表命令

- **#排表规则**
- #新建表
- #设置地点
- #设置规则
- #设置战队
- #查看表
- #结束表

##### 轮次命令

- %s+ vs %s+ （添加单局战斗）
- #记录 %s+ %d%d
- #下一轮
- #切换第%d 轮
